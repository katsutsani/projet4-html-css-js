Environnement linux

Lors de se projet, l'un des objectifs était de développer sur un environnement linux. Pour se faire nous avons utiliser Debian. Au début le plus difficile était de se déplacer dans les fichiers car Debian n'as pas d'interface graphique et toute action se fait a partir de ligne de commande

Github

Afin de travailler en groupe, on nous à demander d'utiliser git et github et de nous créer un environnement professionnel afin de stocker notre travail et de pouvoir se mettre à jour lorsque l'un de nous faisait une modification

Puissance 4

Pour la partie programmation, le projet consistait à faire un puissance 4 pour se faire, il nous à fallut un éditeur de texte type emacs puis il nous as fallut utiliser gcc afin de compiler notre code pour pouvoir le lancer

Découverte d'outils de travail

C'est toujours un plaisir d'apprendre de nouveau langage de programmation même si nous faisons facilement des erreurs, on apprend à ne plus les refaires. Mais c'est encore plus plaisant lorsque qu'on apprend en même temps de nouveau outils de travail comme ca été le cas pour se projet avec la découverte du C, de l'éditeur emacs mais aussi de trello pour partager les taches et de github pour mettre le travail en commun

flash_on

Organisation du travail

Pour pouvoir bien avancer, il faut bien organiser le travail et ce projet nous à bien aider pour apprendre comment s'organiser grace à trello pour partager les taches à faire mais aussi grace a github qui nous permettait de mettre à jour nos fichier facilement quand une modification était apporté
